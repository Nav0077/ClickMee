## üöÄ Overview

**ClickMee** is a next-generation competitive clicker game set in the void of space. Featuring a stunning **Cosmic Dashboard** aesthetic, real-time global leaderboards, and a satisfying "Reactor Core" interaction mechanic.

Compete against the world to reach the **1 Trillion** score cap and dominate the void.

---

## ‚ú® Features

-   **Cosmic HUD Interface**: A fully immersive, glassmorphism-based UI with floating stats and particle effects.
-   **Reactor Core Mechanic**: A dynamic, multi-layered button that pulses and rotates with every interaction.
-   **Global Leaderboard**: Real-time ranking system powered by Supabase.
-   **Identity Sync**: Automatically syncs your Google profile name and avatar (or customize your own).
-   **Massive Scale**: Supports scores up to **1,000,000,000,000** (1 Trillion).
